About

Installation

  • Copy the zip archive and unzip
  • In your html header, include:
    <link href="lib/simpleLightbox.css" rel="stylesheet" type="text/css" />
    <script src="lib/simpleLightbox.js">

Use

In each image you include in your HTML, add a call to the lib:
<img src="png/02.png" height="100" value="TEST LIGHTBOX" onclick="simpleLightbox(theURL,thePadding,showTitle);"> The simpleLightBox function takes 3 parameters:
theURL:
A relative or absolute path to the image file (same as in the 'src' parameter of your IMG tag)
thePadding:
Padding around the image in pixels
showTitle:
Boolean that determines if title is shown. The title includes the path and size of the original image, as well as the proportion at which it is shown (max 100%) on screen
